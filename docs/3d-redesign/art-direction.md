# Step 12 Art Direction

## Goal

Transform the current stable 2D AI/ML portfolio into a premium cinematic 3D experience while preserving recruiter readability, semantic HTML, accessibility, SEO, mobile usability, and production reliability.

## Visual Principles

- The 3D layer supports the content; it never replaces the content.
- The current dark palette remains the foundation.
- Orange remains the primary accent because it already anchors the brand.
- Blue/cyan may appear only as a secondary data-signal color.
- Typography remains HTML, not baked into canvas textures.
- Motion is slow, legible, and interruptible.
- The central scene must communicate engineering precision rather than generic sci-fi decoration.

## Concept A - Neural Intelligence Core

An illuminated AI core sits behind and beside the hero content. It is built from procedural geometry: a central processor-like lattice, neural/data nodes, and thin connection paths. Scrolling moves the camera through layers of the system: identity, capabilities, projects, research, experience, and contact.

| Criterion | Evaluation |
|---|---|
| Reference fidelity | Cannot be measured against unavailable references; fits the requested cinematic AI/ML direction. |
| Originality | High; can be tailored to Veerababu's AI, LLM, data, and research profile. |
| Recruiter readability | High if HTML content remains primary. |
| Technical feasibility | High with procedural Three.js/R3F geometry. |
| Mobile feasibility | High if simplified to static/poster plus a small animated core. |
| Accessibility | High if canvas is decorative and paired with semantic sections. |
| Performance cost | Moderate and controllable with instancing and lazy loading. |
| Asset requirements | Low; mostly procedural. |
| Implementation risk | Moderate; scroll/camera orchestration needs discipline. |
| Section coverage | Strong; nodes can map naturally to projects, research, experience, and contact. |

## Selected Concept

**Concept A - Neural Intelligence Core** is the primary direction.

## Why This Concept Wins

- It best represents artificial intelligence, machine learning, distributed systems, data pipelines, research, and precise engineering in one coherent scene.
- It can be built procedurally, avoiding large unlicensed model assets.
- It supports a strong hero identity without turning the portfolio into a game.
- It maps cleanly to the existing sections:
  - Hero: central intelligence core.
  - About/capabilities: labeled capability circuits.
  - Projects: connected modules around the core.
  - Research: publication artifacts as data planes.
  - Experience/education: timeline path through the system.
  - Contact: final stable terminal/connection node.
- It has the best chance of preserving performance because Step 13 can prototype a small procedural scene before adding assets.

## Visual Language

| Layer | Direction |
|---|---|
| Scene base | Deep charcoal void with subtle grid and depth haze. |
| Core object | Procedural processor/neural lattice with emissive orange core. |
| Data nodes | Small instanced points and capsules, grouped by portfolio section. |
| Lines | Thin low-opacity connection splines, no dense spaghetti network. |
| Project modules | Minimal glass/metal plaques paired with semantic HTML cards. |
| Research artifacts | Floating diagram planes using truthful project visuals only. |
| Lighting | Ambient light plus emissive materials; avoid expensive shadows initially. |
| Post-processing | None in Step 13; optional bloom later only if performance allows. |
| Typography | Existing HTML typography, slightly layered over/around scene. |
| Mobile | Simplified core poster or low-poly core, no dense node cloud. |

## Features Worth Reproducing From Current Site

- Dark, high-contrast foundation.
- Strong first-screen identity.
- Clear CTAs.
- Orange accent as action signal.
- Project cards with real metrics and links.
- Accessible mobile navigation.

## Features Not To Copy Or Add

- Blank WebGL loading screens.
- Forced multi-second intro animations.
- Custom cursor that hides the native cursor.
- Canvas-only project labels.
- Heavy physics, path tracing, full-screen video textures, or audio.
- Scroll hijacking that prevents normal page control.
